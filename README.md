# CustomPaddleCudaKernel
实现自定义Paddle的CUDA算子，用来学习CUDA以及benchmark和profile

# Usage
## build
```
python setup_cuda.py install
```
## run benchmark
```
cd ../benchmark
python benchmark.py
```
